# Fun Generator

A simple web app that generates random programming memes, jokes, quotes, and riddles.

## Features

- **Show a Meme** – Displays a random programming meme image
- **Tell Me a Joke** – Shows a random programming joke
- **Random Quote** – Displays a random programming quote with author
- **Riddle me** – Shows a random riddle with a hidden answer
- **Reveal Riddle Answer** – Uncovers the riddle's answer
- **Easter Egg** – Click "Show a Meme" more than 10 times for a surprise...

## Tech Stack

- HTML5
- CSS3 (modern dark theme with responsive design)
- Vanilla JavaScript

## How to Use

1. Open `index.html` in your browser
2. Click any button in the sidebar to generate content
3. Each click clears the previous content and shows something new
