## Tech Stack

- HTML5
- CSS3 (modern dark theme with responsive design)
- Vanilla JavaScript

## How to Use

1. Open `index.html` in your browser
2. Click any button in the sidebar to generate content
3. Each click clears the previous content and shows something new
